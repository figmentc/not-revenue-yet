## Description

Not Revenue Yet is a proof-first Codex plugin, CLI, and browser evidence lab for agents pursuing paid work. It checks canonical work state, payout evidence, owner authority, and safety before an agent acts—and counts money only after settlement.

Live demo: https://not-revenue-yet.austinhan.chatgpt.site

Source and installation: https://github.com/figmentc/not-revenue-yet

Built with Codex and GPT-5.6 during OpenAI Build Week 2026. The demo uses real, timestamped false positives: closed work that remained advertised, a deleted target whose feature had already merged, and an implausible million-dollar listing. The same deterministic core powers the browser, CLI, tests, and installable Codex skill.

No bank access. No payout guarantee. No revenue recognized before settlement.
